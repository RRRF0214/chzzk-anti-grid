---

## 🛠️ Installation Steps

1. [파일 다운로드](https://github.com/RRRF0214/chzzk-anti-grid/archive/refs/heads/main.zip) 후 폴더에 압축해제  
2. Chrome 주소창에 아래 경로를 복사해서 붙여넣으세요:

    ```
    chrome://extensions
    ```

3. 우상단 '개발자 모드' ON  
4. '압축해제된 확장 프로그램을 로드합니다.' 선택  
5. 프로젝트 폴더 선택
