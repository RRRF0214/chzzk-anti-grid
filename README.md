<h2>🛠️ Installation Steps:</h2>

<p>1. [파일 다운로드] 후 폴더에 압축해제[dl1]</p>

<p>2. [chrome://extensions](chrome://extensions) 접속</p>

<p>3. 우상단 '개발자 모드' ON</p>

<p>4. '압축해제된 확장 프로그램을 로드합니다.' 선택</p>

<p>5. 프로젝트 폴더 선택</p>
